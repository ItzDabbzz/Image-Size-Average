# Image-Size-Average
A PowerShell script to grab the average sizes of the images in a folder, and output them into a txt file.

# Usage
```pwsh
.\sizes.ps1 -folder "C:\path\to\your\folder" -outputFile "C:\path\to\your\output.txt"
```
